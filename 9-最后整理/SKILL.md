# 9-最后整理 — 会话收尾与沉淀

会话收尾仪式，四件事：**修改总结**（本次改了什么）、**经验沉淀**（把教训/决策记进 memory）、**项目结构整理**（清临时文件、理目录）、**交接 + 护栏**。

> **边界**：本 skill 管「本次会话的沉淀 + 物理整理」，动作偏加法。
> 需要**全局文档↔代码洁癖同步、尺寸体检、记忆毕业、消除矛盾**时 → 交给 [`0--neat-freak`]，那是知识库编辑，动作偏减法/校准。
> 判据：只处理本次会话产出 → 留在这里；要审查整个知识库是否腐烂 → 转 neat-freak。

---

## MUST 规则

1. **只沉淀本次会话的产出。** 全局知识库审查不在这里做，转 neat-freak。
2. **经验写进 memory，不写进 CLAUDE.md。** memory=跨会话教训/决策；CLAUDE.md=规则手册。
3. **相对时间转绝对日期。** 写进 memory 的事实用 `2026-07-13` 这类绝对日期。
4. **真的改文件。** 用 Edit/Write 落盘，不描述"我会怎么做"。

---

## 步骤

### 1. 修改总结

回顾本次会话，给一份「改了什么」的摘要——供交接和后续复盘用。

- 本次动了哪些文件 / 模块（可参照 `git diff --stat`）
- 做出了哪些关键决策，为什么
- 什么完成了、什么还没做

### 2. 经验沉淀（知识/经验积累）

把本次会话值得跨会话复用的东西写进 **agent memory**。

**什么该沉淀**：
- 踩过的坑 + 绕过方式（下次别再踩）
- 非显而易见的决策与其理由
- 项目特有的约定、陷阱、外部依赖行为

**什么不该沉淀**：
- 代码结构本身（读代码就知道）→ 不记
- 一次性调试细节、已完成的计划 → 不记
- 项目规则、命令速查 → 属于 CLAUDE.md，不属于 memory

**写法**：一条 memory 说清一件事，事实用绝对日期。若发现同类教训已出现多次、或已稳定成"系统怎么工作"→ 这是**毕业**信号，转 `0--neat-freak` 处理，别在这里硬塞。

### 3. 项目结构整理

物理层面的收尾——让工作区干净，不动知识库内容。

- 清理本次会话产生的临时/试验文件（原型脚本、scratch、调试产物、无用输出）
- 散落在根目录的文件归位到合适目录
- 检查目录结构是否还合理（新增模块有没有放对地方）
- **删除前先确认**：不是你本次产生的、或用途存疑的文件，先问再删

### 4. 会话交接

创建交接文档，让下个会话无缝继续。

**规则**：
- 保存到临时文件，返回路径给你
- 不重复已在其他产物（PRD、commit、diff）中的内容，引用路径或 URL 即可
- 建议下个会话要用到的技能

模板：

```md
# Handoff: <会话主题>

## 当前状态
做到了哪一步、什么还没做。

## 已做出的决策
- [决策 1] — 为什么

## 未决事项
- [待定事项]

## 下一步行动
1. [行动] → 建议使用的技能

## 引用
- PRD: [路径]
- 相关文件: [路径]
```

### 5. 安全护栏确认

检查 `0-安全护栏` 是否已配置并生效。

- [ ] `settings.json` 中 PreToolUse 钩子已注册
- [ ] 钩子脚本存在并可执行

如果尚未配置，询问是否需要安装。

所有步骤完成后，输出摘要：

```

### 修改总结
- 本次改动：xxx

### 经验沉淀

- （若有毕业信号，提示转 0--neat-freak）

### 项目结构整理
- 清理临时文件：xxx
- 归位/调整：xxx

### 交接
- 交接文档 → handoff-xxxx.md

### 安全护栏
- 已配置 / 需安装
```

## 什么时候用

- 完成一个阶段/功能后收尾
- 结束一天的工作前
- 需要把项目交给另一个人或 agent

需要深度审查整个知识库是否与代码一致、记忆是否膨胀 → 用 [`0--neat-freak`]。

## 案例

```
你：今天做完了，帮我收尾
Claude：1. 修改总结 → 改了 payment 模块 3 个文件，决定用原路退款
        2. 经验沉淀 → memory 新增"退款网关有 5% 超时，需重试"
        3. 项目结构整理 → 删掉 scratch_test.py 等 2 个临时文件
        4. 交接 → handoff-a7k3.md
        5. 安全护栏 → 已配置

        注：CLAUDE.md 里 payment 路由表看着有点旧，建议跑一次 0--neat-freak 做全局同步。
```

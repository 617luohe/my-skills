# 画布设计

两步完成：
1. **设计哲学创建**（.md 文件）
2. **在画布上表达**（.pdf 或 .png 文件）

## 第一步：创建设计哲学

### 核心理解
- 输入：用户提供的微妙输入或指令，作为基础但不限制创作自由
- 输出：一个设计哲学/美学运动
- 下一步：同一版本的 Claude 接收哲学并**在视觉上表达它**

### 如何生成视觉哲学

**命名运动**（1-2 词）：如"野兽派喜悦""色彩沉默""新陈代谢之梦"

**阐述哲学**（4-6 段）：
- 空间与形式
- 色彩与材质
- 尺度与节奏
- 构图与平衡
- 视觉层次

**关键准则**：
- 避免冗余，每个设计方面只说一次
- 反复强调工艺：最终作品应展现"精心雕琢""大师级执行""无数小时的打磨"
- 给下一阶段留创作空间：方向具体但不扼杀诠释空间

### 核心原则
- **视觉哲学**：创造一种美学世界观，通过设计表达
- **最少文字**：文字稀疏、仅必需品、作为视觉元素集成
- **空间表达**：思想通过空间、形式、色彩、构图传达——不是段落

## 第二步：视觉表达

将哲学转化为画布作品：
- 90% 视觉设计，10% 必要文字
- 让形式、色彩、构图说话
- 文字作为稀有、有力的姿态——永远不成段

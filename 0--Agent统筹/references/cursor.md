# 外援执行器参考：Cursor Agent

## 使用前探测

首次调用前运行 `cursor-agent --version` 与 `cursor-agent --help`，确认当前版本的无头模式、工作区、模型、会话和自动审批参数。命令参数可能随版本变化，必须以本机帮助为准。

安装和认证应以 Cursor 官方文档为准。检测不到命令或尚未登录时，只报告缺口，不通过不透明脚本代替用户完成认证。

## 分派能力

- 无头任务使用机器可读输出格式。
- 写任务绑定独立 worktree，采用安全操作自动批准、高风险操作暂停的审批级别。
- 只读评审优先使用客户端提供的只读或计划模式。
- 网络和工作区之外的访问默认不授权。
- 工具调用被暂停时，外援应以 `NEED_ADVICE:` 或 `需要决策：` 返回阻塞信息，而不是持续等待。

## 会话

- 如果客户端支持预先创建会话，注册时创建并保存会话标识。
- 每次分派恢复同一持久会话，但一个会话同一时间只运行一个任务。
- 模型可枚举时先验证用户指定值；未指定时使用客户端默认模型。

## 验证与异常

- 记录退出码、最终报告和结构化输出位置。
- 自动审批参数不存在时降级到客户端当前支持的最小权限模式。
- 工作区信任需要交互时停止并说明，不擅自使用完全信任或绕过安全检查。

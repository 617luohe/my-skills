# Excalidraw 图表生成器

从自然语言描述生成专业图表。

## 支持的图表类型

| 用户意图 | 图表类型 | 关键词 |
|----------|----------|--------|
| 流程、步骤、工序 | **流程图** | "工作流""流程""步骤" |
| 连接、依赖、关联 | **关系图** | "关系""连接""依赖" |
| 概念层级、头脑风暴 | **思维导图** | "思维导图""分解""概念" |
| 系统设计、组件 | **架构图** | "架构""系统""模块" |
| 数据流、转换过程 | **数据流图 DFD** | "数据流""数据处理" |
| 跨职能流程、角色 | **泳道图** | "业务流程图""泳道""角色" |
| 面向对象设计 | **类图** | "类""继承""OOP" |
| 交互序列、消息流 | **时序图** | "时序""交互""消息" |
| 数据库设计、实体 | **ER图** | "数据库""实体""数据模型" |

## 工作流程

1. **理解需求**：确定图表类型、关键元素、关系、复杂度
2. **选择类型**：按上表匹配
3. **提取结构化信息**：
   - 流程图：步骤列表、决策点、起终点
   - 关系图：实体/节点 + 关系（从→到，带标签）
   - 思维导图：中心主题 + 主分支(3-6) + 子主题
   - 架构图：组件 + 连接 + 数据流向
4. **生成 JSON**：输出标准 .excalidraw 格式

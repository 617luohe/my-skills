# /生成科研审批单 — 自动生成科研审批单

自动扫描项目根目录下所有 `*年*月报销单/` 文件夹，将里面的报销单 PDF 批量转换为科研审批单 Excel。

## 命令

| 命令 | 作用 |
|---|---|
| `/生成科研审批单 生成` | 自动扫描所有月份，批量生成 |
| `/生成科研审批单 预览` | 预览所有月份数据 |
| `/生成科研审批单 重生成` | 覆盖已有文件重新生成 |

**无需指定参数**，自动识别所有含 PDF 的报销单目录。

### 生成 — 批量生成全部月份

```bash
cd approval-gen && python process_pdfs.py
```

扫描所有 `*年*月报销单/` 目录，逐份生成对应的科研审批单到同名 `*年*月科研审批单/` 目录。

### 预览 — 查看提取结果

```bash
cd approval-gen && python process_pdfs.py --dry-run
```

### 重生成 — 覆盖已有文件

```bash
cd approval-gen && python process_pdfs.py --force
```

## 目录约定

```
2026年5月报销单/       → 2026年5月科研审批单/
2027年1月报销单/       → 2027年1月科研审批单/
```

输入目录名格式：`{年份}年{月份}月报销单`
输出目录自动对应：`{年份}年{月份}月科研审批单`

## 文件名

输出格式：`科研审批单M.D-M.D--姓名.xlsx`（与原文件一致）
